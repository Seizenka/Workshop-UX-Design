# Création du fichier

## Sur Linux

1. Ouvrez PlayOnLinux
2. Excécutez Photoshop
3. Fichier
  * Nouveau ou Ctrl + n
4. Une nouvelle fenêtre s'ouvre, choisir un titre "UI Design desktop"
5. Paramètres prédéfinis
  * Choisir "web"
  * Largeur : 1920 px
  * Longueur : 1080 px
  * résolution : 300

  > Un calque de base est présent, renommez-le "background"
